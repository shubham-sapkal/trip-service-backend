
### Vehicle API


```http
  POST /vehicle/register
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `registrationNumber` | `string` | **Required**. Vehicle Registeration Number |
| `ownerUserName` | `string` | **Required**. Owner Of Vehicle |
| `noOfSeats` | `string` | **Required**. No. Of Seats in Vehicle |
| `pricePerKm` | `number` | **Required**. Price Per Km |
